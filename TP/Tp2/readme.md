## Jeu du juste au prix

Math Random : https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/random 

Le but de ce TP est de réaliser un programme qui génère un prix aléatoire en 1 et 1000 et qui demande à l'utilisateur de deviner le prix généré.
A chaque proposition le programme indiquera si le prix est en dessous, ou au dessus du prix généré.

Dans un deuxième temps on ajoutera un compteur de nombre de tentative.



## Jeu du Pendu

Le but de cd TP est de réaliser un programme qui choisit un mot aléatoire parmi une selection de mots.
- Le programme proposera à l'utilisateur un masque du mot sous la forme(****)
- A chaque proposition de l'utilisateur réussi, le programme affichera le ou les caractères touvés dans le mot, si la proposition est erronée le programme diminuera le nombre de tentative restante.
